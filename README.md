# Sales Data Analytics and Prediction Project

## Overview
This project analyzes sales data using Python.  
It includes:
- Data Cleaning
- Data Visualization
- Business Insights
- Machine Learning Sales Prediction

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Sales by Category Analysis
- Profit by Region Analysis
- Yearly Sales Trend
- Category Distribution Pie Chart
- Sales Prediction Model

## Machine Learning
Linear Regression model is used to predict future sales based on:
- Quantity
- Discount
- Shipping Cost

## Project Structure
- analysis.py
- graphs/
- dataset

## Output
Graphs are saved inside the graphs folder.
